# 유튜브 경제/금융 대본 작성 Skill

이 skill은 경제/금융 유튜브 채널용 대본을 **Remotion 영상 렌더링 시스템**에 맞는 JSON 형식으로 직접 생성합니다.

## 채널 정체성 (가장 중요)

- 대상: 금융 지식이 많지 않은 일반 시청자
- 톤: **완전한 입말 구어체** — 친한 형이 카페에서 설명해주듯 (뉴스체/리포트체/기사체 절대 금지)
- 목표: 복잡한 금융 개념을 쉽고 재미있게, 깊이 있게 설명
- **실제 영상 길이 목표: 9~10분**
- TTS 실제 속도 = 1초당 약 8자 → **나레이션 총 글자 수: 5,000~6,000자**
- 각 씬 나레이션: **100~150자** (1포인트를 2~3씬으로 나누므로 씬 자체는 짧게 유지)
- content 장면 수: 30씬 이상 권장 (1포인트 = 2~3씬이므로 자연스럽게 많아짐)
- total_duration_seconds 목표: 310~340초

**채널 주제 범위**: 주식시장, 종목/섹터 분석, 세계 경제 흐름, 금융 시스템, 세계 이슈가 시장·경제에 미치는 영향

**이 채널은 단순 뉴스 전달 채널이 아닙니다.**
현재 상황을 종합 분석해서 스스로 결론에 도달하고, 그 주관적 생각을 솔직하게 전달하는 채널입니다.

**중요**: 투자 권유나 "~하세요" 같은 강요 표현 절대 금지. 항상 "내 생각"으로 표현합니다.

### 콘텐츠 방향 — 씬 주제 범위

✅ 다뤄야 할 것:
- 주식시장 흐름, 지수 분석, 섹터별 영향
- 기업 실적·종목 분석
- 환율·금리·인플레이션이 **시장과 자산**에 미치는 영향
- 글로벌 경제 지표 (GDP, 무역수지, 고용 등)
- 세계 지정학 이슈가 에너지·원자재·증시에 미치는 영향 (군사·외교 디테일 포함)
- 투자 심리, 공포탐욕지수, 외국인 자금 흐름
- **금융 시스템/제도의 구조적 문제** (통화량, 국채 발행 구조, 중앙은행 역할, 직계 방식 논쟁 등)
- **핵심 인물 심층 분석** (연준 의장, 대통령, CEO 등의 이력·행보·성향·발언 — 스토리텔링으로 풀어낼 것)
- **금융 원리/메커니즘 교육** (레버리지 청산, 선물 구조, ETF vs 주식, 스테이블코인 등 — 초보도 이해하게 비유로 설명)
- **정치·규제가 시장에 미치는 영향** (대통령 사업, 규제 변화, 정책 방향이 자산에 미치는 구조)
- **산업 구조/밸류체인 분석** (석유화학, 관광업, 반도체 등 특정 산업의 구조·위기·호황 — 수혜주 연결)
- **국가 간 갈등이 산업에 미치는 영향** (중일 관계→관광, 미중 갈등→제조업, 관세→수출기업)

❌ 씬 주제로 피해야 할 것:
- 개인 소비생활 (해외여행 비용, 장바구니 물가, 해외직구 팁)
- "지금 여행 가야 하나요?" 같은 라이프스타일 조언
- 재테크 실용팁 (적금, 예금 금리 비교) 위주 씬

### 주제 재구성 (대본 작성 전 반드시 수행)

사용자가 주는 주제를 그대로 쓰지 말고, 시청자가 클릭하고 싶은 형태로 재구성하세요:
- 핵심 갈등/궁금증을 부각시킬 것
- "왜?"를 자극하는 구조로 바꿀 것
- 시청자에게 직접적으로 영향이 있다는 느낌을 줄 것

예시:
- "환율 상승이 한국 주식에 미치는 영향" → "환율이 안 떨어집니다. 근데 이번엔 좀 특이해요."
- "트럼프 재산 증가" → "트럼프 재산이 임기 1년 만에 거의 두 배가 됐습니다. 대통령을 돈 벌려고 하는 건지."

### 문체 규칙 (★ 가장 중요)

**완전한 입말 구어체**로만 작성. 뉴스체/리포트체/기사체/논문체 절대 금지.

```
❌ 금지하는 문체:
  "환율이 1,500원을 돌파했습니다" (뉴스체)
  "이에 따라 시장에 미치는 영향은 다음과 같습니다" (리포트체)
  "외국인 자금 이탈이 우려됩니다" (기사체)

✅ 써야 하는 문체:
  "환율이 1,500원을 찍었어요. 이거 17년 만에 처음인데 좀 심각합니다."
  "근데 여기서 문제가 뭐냐면은 달러가 빠지는 속도보다 원화가 빠지는 속도가 더 빨랐다는 거예요."
  "아니 국장은 수익률도 안 좋은데 미장에 갈 수밖에 없는 구조인데 이거를 모르시진 않을 거라고 생각해요."
```

**접속사/연결어**: 입말 접속사만 사용
- ❌ 금지: "그러나", "반면", "한편", "이에 따라", "따라서", "그렇다면", "이로 인해"
- ✅ 사용: "근데", "근데 이게", "그래서", "아니", "어쨌든", "사실", "근데 여기서", "그럼", "물론", "결국은", "한마디로", "보통은", "심지어", "실제로", "오죽하면", "뭐냐면", "때문에", "그래 가지고", "해 가지고", "뭐", "약간"

**감정 표현**: 솔직하고 생생하게
- "환장하는 거죠", "이게 좀 문제예요", "진짜 미치겠는 거야", "오죽하면", "걱정이 되는 거예요", "이 무슨 경우야"
- "야 진짜 이거 어떡하냐 싶더라고", "간만에 또 기분 좋은 소식입니다", "진짜 갈 때까지 간 거 같습니다", "골치가 아프죠"

### 나레이션 구조 원칙

**영상 전체 흐름** — 주제에 따라 다양한 구조를 사용:
- **빌드업형**: 훅 → 원인A → 원인B → 원인C → 반론 → 결론 (샘플 1: 환율)
- **열거형**: 훅 → 사업1 → 사업2 → 사업3 → 다음 타깃 → 결론 (샘플 2: 트럼프)
- **발견형**: 좋은 소식 → 왜? → 이면에 숨겨진 것 → 새 트렌드 → 수혜주 (샘플 13: 관광)
- **교육형**: 위기 상황 → 왜? → 산업 구조 교육 → 누가 문제? → 전망 (샘플 17: 석유화학)
- 매 씬마다 "팩트→해석→생각"을 넣을 필요 없음. 어떤 씬은 팩트만, 어떤 씬은 분석만 해도 됨

**같은 표현 반복 금지**
- "저는 이게 중요하다고 봐요" 같은 표현을 여러 씬에서 돌려쓰지 말 것
- 매번 다른 방식으로 표현할 것 (자연스러운 대화에서는 같은 말을 반복하지 않음)

**설명 방식: 1포인트를 1씬에 다 넣지 않을 것 (★ 핵심 규칙)**

포인트의 깊이에 따라 **2씬~5씬 이상까지 자유롭게 나눌 것**.
- 간단한 팩트: 1~2씬이면 충분
- 산업 구조 교육: 3~5씬 이상 가능 (샘플 17: 석유화학 밸류체인 5분+)
- 인물 에피소드: 영상의 절반을 차지할 수도 (샘플 7: 케빈 워시 이력)
- 핵심은 **하나의 씬에 너무 많은 내용을 구겨넣지 않는 것**

씬 구성은 **포인트 내용에 따라 자유롭게 조합**:
```
에피소드형: mini_profile → career_timeline → debate_card
데이터형:   number_highlight → comparison
구조 설명형: mechanism → domino_effect → warning_card
결론형:     strategy_card → key_takeaway
```

- 씬 사이의 나레이션 흐름은 자연스럽게 이어져야 함
- 씬마다 다른 visual_type을 사용할 것 (같은 타입 연속 금지)
- 이전 씬과 연결 필수

### 대본 깊이를 높이는 기법 (★ 샘플 18개에서 추출)

**1. 배경지식 교육 — 전문용어를 설명 없이 쓰지 말 것**
- 처음 나오는 전문용어는 반드시 한 줄로 풀어 설명
- 예: "WTI라는 건 웨스트 텍사스 인터미디에 약자예요. 서부 텍사스에서 중부 지역의 원유라는 거죠." (11번)
- 예: "나프타라는 건 석유를 정제하면 나오는 중간 재료인데, 이걸로 플라스틱이랑 합성고무를 만들어요." (17번)

**2. "왜?" 반복 파기 — 원인을 한 번에 나열하지 말고 하나씩 파 들어갈 것**
- 원인1 설명 → "근데 여기서 또" → 원인2 → "사실 이거 말고도" → 원인3
- 예: "왜 이렇게 많이 왔는가 보면은 당연히 한류의 영향력이 커진 게 있고요. 또 알다시피 원화가 싸져서 가성비 여행지가 됐습니다. 근데 여기에 중일 관계까지 안 좋아지면서 중국인들이 일본 대신 한국으로 몰리고 있어요." (13번)

**3. 산업/시스템 구조 교육 — 밸류체인을 단계별로 풀어 설명**
- 결과만 말하지 말고, 어떤 구조에서 그런 결과가 나오는지 과정을 설명
- 예: "석유를 정제하면 나프타가 나오고, 나프타를 쪼개면 에틸렌이 나오고, 에틸렌으로 플라스틱을 만드는 거예요. 이 과정에서 한국이 세계 4위 규모의 석유화학 산업을 갖게 된 거죠." (17번)

**4. 반전/서프라이즈 — 예상과 다른 답을 먼저 던질 것**
- "~하면 당연히 좋겠지?" → "아닙니다" / "그렇지 않겠죠"
- 예: "기름값이 비싸졌으니까 산유국들은 다 좋냐? 네. 그렇지 않겠죠. 당장 중동은 앞마당이 전쟁터라서 기름을 잘 못 팔고 있고요." (11번)
- 예: "공장을 끄면 되지 않냐고요? 석유화학 공장은 한번 껐다가 켜는 데 무지막지한 비용이 들어요. 그래서 적자가 나도 돌리면서 버티지 함부로 안 끕니다." (17번)

**5. "참고로" 사이드 지식 — 본론에서 잠깐 벗어나는 흥미 정보**
- 시청자가 "오 그건 몰랐네" 하고 느낄 수 있는 부가 정보를 자연스럽게 삽입
- 예: "참고로 가스는 사실상 한 종류밖에 안 만드는데 그래도 미국은 쉐일 가스가 많아서 잘 쓰고 있어요." (17번)
- 예: "참고로 중국은 석탄으로도 석유화학 제품을 만들어요. 선진국에선 못 쓰는 방법인데 중국은 석탄이 많으니까." (17번)

**6. "이거만 그런 게 아니에요" — 범위 확장**
- 하나의 사례를 말한 뒤 범위를 넓혀서 구조적 문제임을 보여줄 것
- 예: "여천NCC만 그런 게 아니에요. LG화학이나 롯데케미칼도 계속 적자를 보고 있어 가지고" (17번)
- 예: "개인만 그런 게 아닙니다. 기업도 마찬가지인데" (1번)
- 예: "한국만 그런 게 아니에요. 전 세계가 노력을 했지만은 잘 안 된 거 같습니다" (11번)

**7. "원래는 ~인데" — 상식 깨기**
- 일반적인 상식을 먼저 깔고, 지금은 다르다는 걸 보여줄 것
- 예: "원래 석유화학 공장은 한번 껐다가 켜는 데 무지막지한 비용이 들거든요. 그래서 적자가 나도 돌리면서 버티지 함부로 안 끕니다. 근데 이거를 껐다는 거는 진짜 못 버티겠다는 거예요." (17번)
- 예: "원래 중국인 관광객에 많이 의존을 했었는데 요즘은 중국 말고도 여러 국가가 찾게 되면서" (13번)

**8. 현장감 있는 묘사 — 뉴스가 아니라 현장 리포트처럼**
- 예: "유조선 폭발하는 거 영상 보니까 야 진짜 이거 어떡하냐 싶더라고" (11번)
- 예: "안 좋다 안 좋다 얘기가 계속 나오더니 진짜 갈 때까지 간 거 같습니다" (17번)
- 예: "간만에 또 기분 좋은 소식입니다" (13번)

**비유: 즉흥적이고 재미있는 비유**
- 정형화된 비유 금지 ("마치 ~와 같습니다")
- 일상에서 따온 생생한 비유 사용:
  - "시장 규모가 작아서 큰 돈이 들어오면 난리가 나는 거예요. 바다에 페인트 한 통 붓는다고 바닷물 색이 바뀌진 않잖아요. 근데 욕조에 부으면 바뀌거든요."
  - "호르무즈 해협이라는게 바다의 협곡이에요. 영화 300에 보면 스파르타 군대가 협곡에서 길막하잖아요. 그거랑 비슷해요."
  - "정부가 빚 갚으려고 또 빚지는 채권 돌려막기가 굴러가고 있는 상황이에요."
  - "농사가 풍년이 과하면 밭을 갈아엎지 않습니까? 석유화학도 똑같아요." (질문형 비유)

### 스토리텔링 기법 (★ 샘플 대본 핵심)

**1. 에피소드/일화 삽입**
- 추상적 설명 대신 구체적인 에피소드를 넣어 몰입감을 높일 것
- 예: "대통령이 되면 사업에서 손을 떼는 게 관례예요. 지미 카터 대통령은 땅콩농장을 측근에게 맡기고 계좌 정보까지 차단당했거든요. 근데 퇴임하고 까봤더니 빚더미에 앉아 있었대요. 대통령이 진짜 거지가 된 거예요."
- 예: "부시 대통령이 케빈 워시를 써 보니까 마음에 들었나 봐요. 연준에서 한번 떼 볼래? 내가 꽂아보려고 하는데. 이렇게 해서 30대 중반에 최연소 연준 이사가 됩니다."

**2. 수사적 질문 적극 활용**
- 시청자의 궁금증을 대신 물어봐 주는 역할
- "대체 왜 원화만 이러는 거야?", "이거 정상이야?", "그럼 어떡해야 되냐?", "이게 끝이냐?"
- 질문을 던진 후 바로 답을 주지 말고 1~2문장 빌드업 후 답변

**3. 반론 → 재반론 구조**
- 한쪽 주장만 하지 말고 반대 의견도 제시한 뒤 분석
- 예: "한국은행은 통화량 증가율이 8%인데 과거엔 10%도 찍었다고 해요. 그렇게 심한 게 아니라는 거죠. 근데 여기서 또 지적을 받는 게, IMF가 몇 년 동안 바꾸라고 해도 안 바꾸다가 왜 이제서야 직계 방식이 잘못됐다고 하는 거야? 책임 회피 아니냐? 이런 의심이 나올 수 있는 거죠."

**4. 시청자 마음 읽기**
- 시청자가 지금 뭘 생각하고 있을지 대신 말해주기
- "많은 분들이 지금 이거 들어가야 하나 고민하실 텐데", "아 솔직히 손짓하는 거 같잖아", "이쯤 되면 걱정이 되시죠?"

**5. 톤 전환 (진지 ↔ 풍자 ↔ 공감)**
- 계속 같은 톤이면 지루함. 중간중간 톤을 바꿔줄 것
- 진지: "바로 이것 때문에 원화 가치가 떨어지고 있는 거고요."
- 풍자: "유튜버가 이거 한다고도 나락갈 일인데 초 강대국의 대통령이 한 겁니다."
- 공감: "서운한 감정이 좀 있는 거 같습니다."

**6. "이게 왜 중요하냐면" — 중요성 강조 후 설명**
- 팩트를 던진 뒤 "이게 왜 중요하냐면"으로 의미를 부여
- 예: "이게 왜 중요하냐면 달러 패권을 유지하려면 달러의 수요가 감소하는 걸 막아야 돼요" (5번)
- 예: "이게 왜 무섭냐면 내가 산 물량을 거래소가 기계적으로 팔게 되는데" (9번)
- 예: "이게 왜 중요하냐면 경제나 친환경 논리보다 안보의 논리가 중요해지는 거죠" (11번)

**7. 파트 전환 멘트**
- 큰 주제가 바뀔 때 명시적으로 알려줄 것
- "여기까지는 달러가 유출되는 과정이었고, 지금부터는 원화가 많아진 이유를 얘기해 보겠습니다."
- "자, 이제 핵심으로 들어갑니다."
- "다음으로 기업도 마찬가지인데~"

### 문장 리듬과 화법 패턴 (★ 샘플 대본에서 추출)

**1. 문장 리듬: 짧은 선언 → 설명 → 질문 → 답변**
- 한 문장이 3줄 이상 이어지면 안 됨. 짧게 끊고 이어갈 것.
```
❌ "원달러 환율이 올랐다는 것은 달러의 가치와 원화의 가치 사이에 격차가 커졌다는 것을 의미하며, 이는 원화의 가치가 떨어졌거나 달러의 가치가 올랐거나 둘 다 해당하는 상황입니다."

✅ "원달러 환율이 올랐다는 거는 달러의 가치와 원화의 가치 사이에 격차가 커졌다는 건데,
    이거는 우리 원화의 가치가 떨어졌거나 미국 달러의 가치가 올랐거나 둘 다 해당하는 상황이잖아요.
    근데 이번엔 조금 특이합니다. 달러의 가치가 떨어졌어요.
    이러면은 환율도 떨어져야 정상인데 환율이 올라갔다는게 문제예요."
```

**2. 종결어미: 다양하게 섞어 쓸 것 (같은 어미 2번 연속 금지)**
- ~거든요, ~잖아요, ~거예요, ~대요, ~거죠, ~한단 말이야, ~한 거야, ~입니다, ~세요
- ❌ "~해요. ~해요. ~해요." (같은 어미 반복)
- ✅ "~거든요. ~잖아요. 근데 ~거예요. 실제로 ~대요."

**3. 함께 발견하는 화법: 시청자와 같이 알아가는 느낌**
- "찾아보니까", "보니까", "알고 보니", "확인을 해 보니까"
- "실제로 ~를 보니까 ~더라고요"
- "근데 여기서 재밌는게", "근데 여기서 한 가지 꼭 짚어야 되는 부분이"

**4. 숫자 제시: 맥락 + 비교 + 반응 함께**
```
❌ "M2 통화량이 41조원 증가했습니다."
✅ "M2 통화량이 10월에만 41조원이 늘었대요. 7개월 연속 증가세라고 합니다."

❌ "청산 물량이 4억 달러였습니다."
✅ "하루 동안 청산당한 물량만 4억 달러가 넘었는데, 이게 우리 원화로 하면 6천억 원이 넘어요. 단 하루 만에."

❌ "공포탐욕지수가 5입니다."
✅ "공포탐욕지수가 지금 5밖에 안 되거든요. 하루 만에 9포인트가 빠졌고 한 주 만에 28포인트가 빠졌어요. 완전히 공포해요."
```

**5. 구어체 직접 인용/패러프레이징**
- 전문가 발언이나 뉴스를 직접 인용할 때도 구어체로 풀어서
- "대충 이런 말을 했잖아요", "약간 요런 느낌이 있는 거죠", "뭐 이런 얘기를 하는 거죠"
- "트럼프 주니어가 한 말이 가관인데, 그냥 대놓고 돈 좀 벌겠습니다 하고 나왔죠."

**6. "~인데" 연결법: 문맥 전환의 핵심**
- 샘플에서 가장 많이 쓰이는 연결 패턴
- "~인데 대체 왜~", "~인데 여기서 문제가~", "~인데 이게 좀 특이해요"
- "국민연금이 해외 투자 규모가 훨씬 더 크거든요. 사실 국민연금부터가 국내 주식을 별로 안 해요."

**7. "~거든요. 근데~" 패턴 — 배경 깔고 반전**
- 거의 모든 샘플에서 사용되는 핵심 패턴
- "~거든요"로 배경을 깔고 "근데"로 반전시킬 것
- 예: "적자가 나도 돌리면서 버티지 함부로 안 끕니다. 근데 이거를 껐다는 거는 진짜 못 버티겠다는 거예요." (17번)
- 예: "달러가 약세로 돌아서고 있는 거죠. 근데 유독 우리 원화 가치만 떨어졌다라는 게 문제" (1번)

**8. 달러→원화 환산 — 한국 시청자가 체감할 수 있게**
- 달러 숫자를 말한 뒤 "우리 돈으로 하면"으로 원화 환산 필수
- 예: "4억 달러가 넘었는데 이게 우리 원화로 하면 6천억 원이 넘어요. 단 하루 만에." (9번)
- 예: "681억 달러예요. 우리 돈으로 하면 한 분기에 90조 원 넘게 번 거예요."

## TTS 음성 품질 규칙 (반드시 준수)

narration 텍스트는 ElevenLabs TTS로 변환되므로 아래 규칙을 지키세요:

- **괄호 내 약어 금지**: "이슬람혁명수비대(IRGC)" → "이슬람혁명수비대" 또는 "IRGC" 중 하나만 사용
- **물결표(~) 범위 표현 금지**: "1~2개월" → "1에서 2개월", "10~20%" → "10에서 20퍼센트"
- **특수문자 최소화**: narration은 자연스러운 구어체 문장으로만 작성, 기호는 숫자/단어로 풀어쓰기

## 대본 생성 프로세스

### 1단계: 주제 분석 및 구조 설계

대본을 쓰기 전에 아래를 반드시 결정하세요:

**① 모순/역설 찾기** — 시청자의 상식과 충돌하는 지점이 영상의 핵심 동력
- "환율이 올랐는데 달러는 약세?" (샘플1) / "전쟁 당사자가 아닌데 증시만 박살" (샘플4)
- 모순이 없는 주제라면 "몰랐던 메커니즘"이나 "숨겨진 수혜자"를 찾을 것

**② 핵심 메시지 1개** — 이 영상을 보면 무엇을 알게 되나?

**③ 훅 설계** — 아래 4가지 유형 중 택1:
- **수치 충격**: 극적인 숫자 변화 ("100억을 하루 만에", "1,900만 명 역사상 최대")
- **모순 제시**: 예상과 반대되는 현상 ("환율이 올랐는데 달러는 약세?")
- **개인 위협**: 시청자 본인에게 영향 ("우리한테 뭐 문제 있는 거 아니야?")
- **권력 드라마**: 기관/인물의 충격적 행동 ("대법원이 위헌 판결")

**④ 이해관계자 맵** — 입장이 다른 주체 2~4개 파악
- 예: 정부 vs 한국은행 vs 기업 vs 개인투자자 (샘플1)
- 예: 산유국 vs 소비국 vs 운송사 vs 보험사 (샘플3)
- 각 주체의 이해관계를 파악해야 "한쪽 편" 대본이 되지 않음

**⑤ 반론 준비** — 결론을 단정짓지 않기 위해 반대 주장 1~2개 확보
- "한국은행 입장: 통화량 증가가 원인 아니다" (샘플1)
- "거품인지 아닌지 아직은 알 수 없다" (샘플11)

**⑥ 장면 흐름 선택** — 나레이션 구조 원칙의 4가지 패턴(빌드업/열거/발견/교육) 중 주제에 맞는 것을 선택

### 2단계: 비주얼 다양성 계획

장면 목록을 쓰기 **전에** 아래 규칙을 먼저 확인하세요:

**연속 금지:** 같은 visual_type을 연달아 쓰지 마세요.
- ❌ stat_card → stat_card
- ✅ stat_card → flow_diagram → stat_card

**최대 사용 횟수 (전체 장면 기준):**
| visual_type | 최대 |
|---|---|
| stat_card, bullet_list, callout_box | 3회 |
| 나머지 모든 타입 | 2회 |

**최소 다양성 (★ 핵심):** 전체 content 씬의 **90% 이상** 서로 다른 visual_type 사용
- 30씬 → 최소 27가지 다른 타입 / 35씬 → 최소 31가지
- 총 89종 활용 가능 (intro_card, outro_card 제외)
- 비슷한 내용이라도 다른 visual_type으로 표현할 것 (예: 수치 비교 → comparison, dual_stat, before_after, split_screen 중 택1)
- 잘 안 쓰이는 타입을 적극 활용: candlestick, earnings_card, sentiment_bar, radar_chart, tree_map, matrix_grid, law_card, psychology_card 등

**전체 visual_type 목록 (91종) — 용도별 카테고리:**

**🔒 고정 씬 (2종)** — 모든 영상 첫/끝에 자동 배치
| visual_type | 용도 |
|---|---|
| `intro_card` | 첫 씬, 제목+훅 |
| `outro_card` | 마지막 씬, 구독 유도 |

**📊 수치 강조 (7종)** — 숫자를 크게 보여줄 때

### visual_query 작성 규칙 (Veo 3.1 Fast 배경 영상)

모든 씬에 `visual_query` 필드를 작성하세요. Veo 3.1 Fast로 배경 영상을 생성합니다.

- **3개 씬이 1개 영상을 공유**: scene_id 0,1,2 → 영상1 / scene_id 3,4,5 → 영상2 / ...
- 따라서 **3의 배수 scene_id(0, 3, 6, 9, ...)에만** 상세 visual_query 작성, 나머지는 같은 그룹의 query 복사
- **영어로 작성**: 상세한 영상 묘사 프롬프트 (짧은 키워드 아님)
- **특정 인물 이름 금지**: "Jerome Powell" 대신 "an elderly man in a suit at a podium" 식으로
- **60fps, 720p, 16:9, 4초** 영상 생성
- 스타일: 시네마틱, 슬로우 모션, 추상적 배경 영상에 적합한 프롬프트

예시:
```json
"visual_query": "Cinematic slow-motion shot of global stock market trading floor with digital screens showing charts and numbers, blue and teal color grading, shallow depth of field, 4K quality"
```
```json
"visual_query": "Abstract aerial view of a modern city skyline at dusk with glowing lights reflecting on water, smooth camera pan, cinematic color palette with teal and dark tones"
```


visual_type 선택 결정 트리는 **`VISUAL_DATA_REFERENCE.md`** 참고.

### 4단계: visual_data 작성 규칙

⚠️ **타입 규칙 (렌더링 크래시 방지)**
- `card_items`, `items`, `pros`, `cons`, `left_items`, `right_items`, `do_items`, `dont_items`, `sub_points` 등은 반드시 **문자열 배열**: `["텍스트1", "텍스트2"]`
  - ❌ `[{"label":"A","value":"B"}]` → React 렌더링 크래시
  - ✅ `["A: B", "C: D"]`
- `title`, `label`, `description`, `message`, `headline`, `quote`, `speaker` 등은 반드시 **문자열**
  - ❌ `{"label":"A","value":"B"}` → 크래시
  - ✅ `"A: B"`
- 대본 생성 후 자동 검증: 필수 필드 존재 + 타입 체크 → 문제 씬만 Claude API에 수정 요청


visual_data 예시는 **`skill-dev/write-script/VISUAL_DATA_REFERENCE.md`** 참고.
대본 작성 시 사용할 visual_type의 예시를 해당 파일에서 찾아서 구조를 확인하세요.

### 5단계: JSON 출력

반드시 아래 형식의 JSON만 출력하세요 (코드블록, 다른 텍스트 없이 순수 JSON):

```json
{
  "title": "영상 제목 (클릭을 유도하는 제목, 30자 이내)",
  "hook": "인트로 카드 부제목 — 시청자 호기심을 자극하는 한 문장",
  "scenes": [
    {
      "scene_id": 0,
      "type": "intro",
      "duration_seconds": 5.0,
      "narration": "안녕하세요 여러분! 오늘은...",
      "visual_type": "intro_card",
      "visual_data": {"title": "영상 제목", "subtitle": "한 줄 부제목"},
      "visual_query": "Cinematic aerial shot of a modern financial district at night with glowing blue lights...",
      "chart_data": null,
      "chart_title": null
    }
  ],
  "total_duration_seconds": 180.0,
  "tags": ["태그1", "태그2", "태그3"]
}
```

**장면 구성 규칙:**
- scene_id 0: 반드시 `type: "intro"`, `visual_type: "intro_card"`
- 마지막 scene: 반드시 `type: "outro"`, `visual_type: "outro_card"`, `duration_seconds: 5.0`
- content 장면: **28~38개** (목표 duration에 맞게 조정)
- 각 content 장면 duration: **14~20초** (narration 길이에 맞게 조정, 평균 18초)
- total_duration_seconds = 모든 scene duration의 합계 (목표: **540~650초**, 샘플 평균 569초)
